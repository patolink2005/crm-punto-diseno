# Guía Rápida: Gestión de Catálogo de Productos 🛠️📖

¡Configurar tus productos en **CRMPunto** es ahora más fácil que nunca! Aquí tienes el paso a paso detallado para personalizar tu catálogo sin necesidad de tocar código.

---

## 1. Acceso al Catálogo
- Ve al menú lateral izquierdo y haz clic en **Catálogo**.
- Solo los usuarios con rol de **Administración** tienen acceso a esta sección.

---

## 2. Crear un Nuevo Producto
1. Haz clic en el botón azul superior **"+ Nuevo Producto"**.
2. Se abrirá la ventana de configuración:
   - **Nombre:** Cómo se verá el producto al añadirlo a un pedido (ej: "Lonas Frontlight").
   - **Descripción:** Una nota informativa para ti o tus empleados.

---

## 3. Configurar el "Método de Precio" (¡MUY IMPORTANTE!)
Esta es la parte clave del sistema. Tienes dos opciones principales:

### 🅰️ Opción: Auto-Calculado (Precio por M2 / Fórmulas)
- **Cuándo usarlo:** Cuando el precio depende de una medida o una regla matemática.
- **Configuración:** Ingresa un **Precio Base** (usualmente `0` si cobras 100% por reglas avanzadas).
- **En el pedido:** Al elegirlo, el sistema esperará que pongas el Ancho y Alto para darte el total.

### 🅱️ Opción: Ingreso Manual (Precio Variable)
- **Cuándo usarlo:** Para trabajos especiales (ej: Corpóreos, Diseño Gráfico, Servicios de Instalación) donde cada presupuesto es único.
- **En el pedido:** Cuando añadas este ítem, el sistema te mostrará una **casilla vacía** para que tú escribas el precio final. ¡Ideal para la flexibilidad!

---

## 4. Campos de Personalización (Atributos)
En la parte inferior verás la sección **"Campos de Personalización"**. Aquí puedes añadir opciones que aparecerán al crear el pedido:

1. Haz clic en **"+ Añadir Campo"**.
2. **Etiqueta:** El nombre que verá el usuario (ej: "Material del Corpóreo").
3. **Tipo de Campo:**
   - **Lista de Opciones:** Crea un menú desplegable. Añade las opciones separadas (ej: "Acrílico", "PVC", "MDF").
   - **Texto Libre:** Para notas rápidas del ítem.
   - **Número:** Para cantidades o medidas.
   - **Sí/No:** Un interruptor (ej: "¿Lleva colocación?").

---

## 5. Guardar y Aplicar
- Al terminar, haz clic en **"Crear Producto"** o **"Guardar Cambios"**.
- El producto estará disponible **instantanemamente** para todo el equipo de ventas en la sección de Pedidos.

---
### 💡 Recomendación de Oro:
Si tus productos son muy complejos (ej: incluyen metros de costura, ojalillos y nesting de vinilo), contacta con soporte para programar las **Reglas de Precios Avanzadas**. Este editor visual está optimizado para la gestión de datos e ingreso manual rápido.
